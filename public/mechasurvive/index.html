🎮 MECHASURVIVE — ALPHA GDD

Room-Based Roguelite Mech Action Game

1. Game Overview

Genre: Action Roguelite / Room-Based Mech Combat

Core Fantasy: Become an unstoppable ace pilot fighting through a compromised space colony, clearing combat arenas and evolving your mech build with each victory.

Elevator Pitch: A fast-paced roguelite mech battler inspired by Hades meets Gundam — where you clear 400m×400m combat rooms, choose powerful upgrades, and push deeper into enemy-controlled territory.

2. Design Pillars

🔥 Power Fantasy First

  • Medium-to-large enemy counts per room
  • Fast kill cadence
  • Spectacle > realism
  • Player should feel: Weak → Skilled → Dominant → Overwhelming

⚙️ Build Evolution

  • Elemental Mods
  • Arsenal upgrades
  • Stat enhancements
  • Build identity emerges via 3-card reward choices

🧬 Room-Based Progression

  • Clear room → Choose 3 upgrade cards → Move to next room
  • Escalating difficulty
  • Player controls pacing via room selection
  • Clear run structure with decision points

3. Core Gameplay Loop

  1. Enter combat room (400m×400m arena)
  2. Engage enemy waves
  3. Clear all enemies
  4. Room cleared → Present 3 upgrade cards
  5. Choose 1 card → Mech evolves
  6. Select next room → Repeat

4. Player Mech — Nova Frame

Base Abilities

  • Boost Dash
  • Melee Weapon (3-hit combo with wing swords)
  • Beam Rifle
  • Ultimate: Hyper Beam (charges on melee hits)
  • 2 Arsenal Slots (manual activation)

Arsenal Abilities

  • Missile Barrage
  • Turret
  • Proximity Mine
  • Orbital Strike
  • FANG System
  • Ion Charge

5. Enemies

Beam Saber Mech

  • Charge attack
  • Beam spear attack
  • Chase and orbit behavior

Beam Rifle Mech

  • Jet movement at range
  • Beam orb attack

Boss (Puru-2 Style)

  • Massive beam sword slice
  • Missile barrage (ground telegraph + rain)
  • Beam shotgun (5 orbs)

6. Progression Systems

Upgrade Cards (~30 implemented)

  • Elemental Infusions (Fire, Lightning, Frost, Corruption)
  • Core Stats (HP, Damage, Speed)
  • Arsenal Enhancements
  • Mobility Mods

7. Setting — Space Colony

Massive orbital space colony under PMC occupation (Gundam-inspired scale).

  • Colony Plaza (open central areas)
  • Industrial Decks (tight corridors, vertical structures)
  • Hangar Bays (wide open, vehicle debris)
  • Observation Platforms (glass, exposed to space)

8. Alpha Success Criteria

  • ✅ Satisfying combat (impact, feedback, mobility)
  • ✅ Clear room structure (enter → fight → win → upgrade)
  • ✅ Build evolution (3-card choice creates identity)
  • ✅ Power escalation (player feels stronger each room)
